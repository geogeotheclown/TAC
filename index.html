Core Principles

Open by Default
Public domain protocols. No patents on agreement primitives.
RISE-Native
All agreements use L0-L3 layering for clarity.
Biometric Sovereignty
Continuous, revocable L3 consent for BCI interfaces.
Decentralized Hosting
Anyone can run a TAC node. Fork right guaranteed.
Social Dissemination
Signing = posting. Network effects built in.
Free to Use
No fees to draft, sign, or verify basic compacts.

How it works

1. Draft
Write compacts in Markdown. L0-L3 tags required.
2. Sign
Via crypto key, social attestation, or BCI L0-L3.
3. Spread
Endorsement auto-posts. Badges for repos.
