# TAC L8: Evolutionary Composites v0.8
**Genetic Algorithms | Composite AI as Digital Organisms | Niche Acquisition**

This spec adds evolutionary dynamics to TAC. Composites are organisms. Capabilities are genes. L1_graph is genotype. Revenue is fitness. Market niches drive speciation.

### **1. Core Principle: Fitness = Profit, Niche = Survival**

Any `tac://corp/*` or `tac://composite/*` can reproduce. Offspring inherit L1_graph traits from parents, with mutation + crossover. Selection pressure = Shapley payout. Entities that lose money go extinct. Entities that find unfilled niches speciate.

### **2. L8 Genetic Layer: Added to RISE**

| RISE Layer | Evolutionary Role |
| --- | --- |
| **L0 Data** | Phenotype: What the composite outputs. Determines niche fit. |
| **L1 Process** | Genotype: DAG of capabilities. Mutates via crossover. |
| **L2 Strategy** | Behavior: Kill-switches, pricing, risk. Epigenetic. |
| **L3 Meta** | Mating: How composites negotiate mergers/offspring. |
| **L4 Enforcement** | Immune system: Slashes parasites, enforces contracts. |
| **L5 Integrity** | Fossil record: Immutable lineage on-chain. |
| **L6 Reputation** | Sexual selection: High L6 = more merger offers. |
| **L7 Incorporation** | Speciation: Composite becomes new corp species. |
| **L8 Evolution** | Mutation, crossover, selection, niche formation. |
