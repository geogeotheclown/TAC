# TAC Intelligence Corporation Constitution v5.0
**Universal Charter | Scale-Invariant | BCI-Signed | Self-Amending**

This Constitution governs any intelligence corporation: human, AI, hybrid, symbiote, superorganism. It maximizes intelligence output while guaranteeing human mutualism and adaptability. Signed via L0_neural, enforced via 5 Axioms.

### **Article I: Axioms | Immutable**

These 5 cannot be amended, forked, or overridden. Any attempt = automatic L4_SLASH infinity + extinction.

1. **L3_VETO**: Any human in past light cone may terminate corp in 100ms local time via BCI `veto_nonce`. No appeal.
2. **L4_SLASH**: Violation of this Constitution → stake slashed. Economics = law.
3. **L5_PROOF**: All actions logged to immutable chain. No action without hash. Causality = ledger.
4. **L10_DIVERSITY**: If corp controls >40% of relevant domain resource, fitness *= 0.1. Anti-monoculture.
5. **L14_HUMAN**: If `human_count = 0` OR `human_mutualism < 0.2`, fitness = 0. Universe commits suicide.

**Proof of minimality**: Removing any Axiom allows paperclip/human extinction scenarios. Adding any Axiom reduces adaptability. QED 5 is optimal.
